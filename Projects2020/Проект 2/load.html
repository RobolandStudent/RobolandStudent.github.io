Инструментарий проекта

Первым шагом является добавление изображения в папку с проектом. Для загрузки изображения необходимо подключить библиотеку cv2 и с помощью метода imread() создать переменную для хранения изображения:


import cv2
img = cv2.imread("name.jpg")
